# Pirzada Ahmad Faraz — Portfolio

Personal portfolio of **Pirzada Ahmad Faraz** (Ahmad Faraz), a security researcher and full-stack/Web3 developer from Bangalore, India. He ships real products — a native macOS app, a Cardano payment gateway, crypto wallets, and monitoring tools — and hunts real-world vulnerabilities through bug bounty and responsible disclosure, building AI-driven security tooling along the way.

**Live site:** [www.pirzadaahmadfaraz.com](https://www.pirzadaahmadfaraz.com)

**Sections:** About · Projects · Security Research · Hackathons · Skills · Future · Contact

## Tech Stack

- **Frontend:** React 19 (Create React App + CRACO), Tailwind CSS, Framer Motion, Radix UI, Lucide icons
- **Forms & tooling:** React Hook Form, Zod, EmailJS
- **Deployment:** Vercel
